# Asset grafico 3D dell'header

L'immagine dell'header del CV (`header-3d.png`) è stata generata in stile **3D**
con **Higgsfield** (modello `nano_banana_pro` / nano_banana_2, 16:9).

## Come ottenere il file locale `header-3d.png`

L'ambiente di esecuzione cloud blocca il download diretto dalla CDN per policy di
rete, quindi il PNG non è stato committato. Per renderlo locale e avere un CV
completamente offline:

1. Apri questo link nel tuo browser e salva l'immagine:
   https://d8j0ntlcm91z4.cloudfront.net/user_3FXJfADIYA4O0kzsOah1cVhQLMC/hf_20260624_170046_707d7369-2f52-45a3-b3fe-dcd8df8e037a.png
2. Salvala in questa cartella con il nome **`header-3d.png`**.

> Nota: anche senza scaricarla, `cv.html` mostra comunque l'immagine grazie a un
> fallback che punta automaticamente all'URL Higgsfield qui sopra (serve
> connessione internet la prima volta).

## Versione VIDEO animata 3D (`header-3d.mp4`)

Header animato in **3D** generato con **Higgsfield** (modello **Kling 3.0**,
image-to-video, 5s, 16:9, muto). Usato in `cv-video.html`.

Per averlo in locale, scarica il file da questo link e salvalo come
**`header-3d.mp4`** in questa cartella:

https://d8j0ntlcm91z4.cloudfront.net/user_3FXJfADIYA4O0kzsOah1cVhQLMC/hf_20260624_170913_4e250960-2da0-4c10-b743-3ea05f05bd20.mp4

> Anche senza scaricarlo, `cv-video.html` riproduce il video grazie a una
> seconda `<source>` che punta direttamente all'URL Higgsfield qui sopra.
